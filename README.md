## 二、开发工具箱

代码目录：~/repo/tools

- wiz：Fabric应用快速开发工具
- json2yaml：json转yaml
- yaml2json：yaml转json
- application：wiz项目的应用模板
- chaincode：wiz项目的链码模板
- network：wiz项目的网络模板
- scripts：wiz项目的脚本模板
